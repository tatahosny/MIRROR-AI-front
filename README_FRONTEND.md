# 🚀 الدليل الشامل لإعداد وتشغيل وتطوير مشروع Smart Mirror
## (من الصفر وحتى البدء في التعديل)

هذا الدليل مخصص للمطورين الذين يرغبون في تشغيل المشروع على أجهزتهم المحلية وتعديل الواجهة الأمامية (Frontend) ولوحة التحكم (Dashboard) بالتفصيل الممل.

---

## 1️⃣ المرحلة الأولى: تثبيت المتطلبات الأساسية (Prerequisites)
قبل فتح المشروع، يجب أن تتأكد من تثبيت البرامج التالية على جهازك:

1.  **لغة PHP (إصدار 8.2 أو أحدث):**
    *   يفضل تحميل حزمة **XAMPP** أو **Laragon** (الأسهل لويندوز) لأنها تحتوي على PHP و MySQL معاً.
2.  **مدير حزم PHP (Composer):**
    *   حمله من [getcomposer.org](https://getcomposer.org/). هذا البرنامج مسؤول عن تحميل مكتبات Laravel.
3.  **بيئة تشغيل Node.js (إصدار 18 أو أحدث):**
    *   حملها من [nodejs.org](https://nodejs.org/). هذا البرنامج مسؤول عن تشغيل مكتبات الفرونت إند (Vite, Tailwind, AlpineJS).
4.  **قاعدة بيانات MySQL:**
    *   تأتي مدمجة مع XAMPP أو Laragon.

---

## 2️⃣ المرحلة الثانية: إعداد المشروع لأول مرة (Setup)
بعد تحميل كود المشروع وفك الضغط عنه، افتح "Terminal" أو "Command Prompt" داخل مجلد المشروع وقم بتنفيذ الخطوات التالية:

### الخطوة 1: تثبيت مكتبات السيرفر (Backend)
```bash
composer install
```

### الخطوة 2: تثبيت مكتبات الواجهة (Frontend)
```bash
npm install
```

### الخطوة 3: إعداد ملف البيئة (.env)
*   ابحث عن ملف اسمه `.env.example` في المجلد الرئيسي.
*   قم بتغيير اسمه إلى `.env` فقط.
*   افتح الملف وابحث عن `DB_DATABASE` وقم بتغيير الاسم إلى اسم قاعدة البيانات التي ستنشئها (مثلاً `smart_mirror`).

### الخطوة 4: إنشاء مفتاح التشفير (App Key)
هذا الأمر ضروري لتأمين بيانات المشروع:
```bash
php artisan key:generate
```

---

## 3️⃣ المرحلة الثالثة: تشغيل المشروع (Running)

لجعل المشروع يعمل بالكامل، ستحتاج لفتح نافذتين Terminal:

**النافذة الأولى (لتشغيل السيرفر):**
```bash
php artisan serve
```
*سيظهر لك رابط مثل `http://127.0.0.1:8000` افتحه في المتصفح.*

**النافذة الثانية (لتشغيل الفرونت إند المباشر):**
```bash
npm run dev
```
*هذا الأمر ضروري جداً؛ بدونه لن تظهر الألوان والتنسيقات (Tailwind) ولن تعمل التعديلات اللحظية.*

---

## 4️⃣ المرحلة الرابعة: الدخول إلى لوحة التحكم (Dashboard)

لوحة التحكم محمية بـ "Auth"، لذا يجب عليك أولاً:
1.  التسجيل عبر الرابط: `http://127.0.0.1:8000/register`
2.  بعد التسجيل، سيتم توجيهك تلقائياً إلى: `http://127.0.0.1:8000/portal/dashboard`

---

## 5️⃣ المرحلة الخامسة: كيف تبدأ التعديل؟ (The "Boring" Details)

إليك خريطة الملفات التي ستعدلها كـ Frontend Developer:

### أين تجد صفحات لوحة التحكم؟
توجه إلى المسار: `resources/views/portal/`
*   `dashboard.blade.php`: الصفحة الرئيسية للوحة التحكم (الإحصائيات).
*   `reports.blade.php`: صفحة عرض تقارير البشرة.
*   `progress.blade.php`: صفحة تتبع صور الحالة.
*   `locations.blade.php`: صفحة الفروع أو المواقع.

### كيف تستخدم التصميم الموحد (Layout)؟
إذا أردت إنشاء صفحة جديدة بنفس تصميم لوحة التحكم، استخدم هذا الكود في ملف Blade الجديد:
```blade
@extends('layouts.portal') 

@section('content')
    <div class="p-6">
        <h1 class="text-2xl font-bold">عنوان الصفحة الجديدة</h1>
        <p>هنا تضع محتوى الصفحة وتستخدم كلاسات Tailwind للألوان والتنسيق.</p>
    </div>
@endsection
```

### 3. المكونات التفاعلية (AlpineJS)
*   المشروع يعتمد على **AlpineJS** للتفاعلات البسيطة والذكية داخل ملفات Blade.
*   ستجد ملفات الجافا سكريبت في المجلد: `resources/js/`

### أين أعدل الألوان والخطوط (CSS)؟
*   المشروع يستخدم **Tailwind CSS v4**.
*   الملف الرئيسي للتنسيقات الخاصة هو: `resources/css/app.css`
*   يمكنك كتابة كلاسات Tailwind مباشرة داخل ملفات `.blade.php` (مثلاً: `class="bg-blue-500 text-white p-4"`).

### أين توجد الصور؟
*   ضع الصور الجديدة في مجلد: `public/images/`
*   لإظهارها في الكود استخدم: `<img src="{{ asset('images/photo.png') }}">`

---

## 💡 نصائح للمطور:
1.  **لا تغلق `npm run dev`:** طالما أنت تعدل في الفرونت إند، اترك هذا الأمر يعمل ليقوم بتحديث المتصفح تلقائياً فور حفظ أي ملف.
2.  **ملفات الـ Blade:** هي ملفات HTML عادية لكنها تقبل كود PHP وبعض الأوامر الخاصة بـ Laravel (مثل `@if`, `@foreach`).
3.  **التجاوب:** لوحة التحكم مصممة لتكون متوافقة مع شاشات المرايا الذكية الكبيرة، لكنها تدعم الموبايل أيضاً عبر كلاسات Tailwind (مثل `hidden md:block`).
